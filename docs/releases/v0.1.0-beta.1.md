# Codex Update Companion v0.1.0-beta.1

Codex Update Companion is an unofficial macOS menu bar app for checking Codex update information from public sources.

## Highlights

- Shows Codex Mac app status in the macOS menu bar.
- Detects whether the Codex app is running.
- Fetches public GitHub Releases from `openai/codex`.
- Fetches public OpenAI Codex changelog entries.
- Shows current Codex Mac app version, latest Mac app feed version, current CLI version, and CLI update guidance.
- Keeps a local JSON cache so previously loaded updates remain visible after restart.
- Handles partial network failures: GitHub Releases and OpenAI changelog are fetched independently.
- Sends macOS notifications for new releases when the user enables notifications.
- Keeps the Dock icon hidden by default.

## Privacy And Permissions

- Unofficial companion app. Not affiliated with OpenAI.
- Does not request OpenAI account credentials.
- Does not request a GitHub personal access token.
- Does not read or modify Codex app internal files.
- Does not scan user project folders.
- Does not request Accessibility permission.
- Does not request Screen Recording permission.
- Uses only public GitHub Releases, public OpenAI changelog, and the public Codex Mac appcast.
- Runs `codex --version` only to detect the local CLI version, with a 3 second timeout.
